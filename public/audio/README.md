# Audio Assets

This directory contains sound effects and ambient audio for the website.

## Active Audio Files

### Ambient Sounds (Background)
- **ambiance-dark.flac** - Dark theme background ambiance (Spaceship interior)
  - Source: Freesound 715175 by suhuy
  - Used: Dark mode background loop

- **ambiance-light.flac** - Light theme background ambiance (Hibernation unit)
  - Source: Freesound 715177 by suhuy
  - Used: Light mode background loop

- **ambiance-legal.ogg** - Legal pages background ambiance (Laboratory)
  - Source: Freesound 212025 by qubodup
  - Used: Privacy/Terms pages background loop

### UI Sounds
- **hover-sound.flac** - UI hover sound effect
  - Source: Freesound 584186 by unfa
  - Used: Link hover interactions

- **click-sound.wav** - UI click sound effect
  - Source: Freesound 469399 by pocosebas
  - Used: Button click interactions
